# Social-to-Lead Agentic Workflow

This project is developed as part of the Machine Learning Internship assignment for ServiceHive.

## Project Overview
A conversational AI agent that:
- Answers product-related queries
- Detects high-intent users (demo, buy, contact)
- Captures lead details (name, email, company)
- Stores leads in a CSV file

## Tech Stack
- Python
- Google Colab
- Pandas

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Chat with the assistant
4. Type 'exit' to stop

## Sample Conversation

User: What is the price?  
Assistant: Our pricing starts from $29 per month.  

User: I want a demo  
Assistant: Please share your details.  
Lead saved successfully.

## Output
Captured leads are stored in leads.csv
